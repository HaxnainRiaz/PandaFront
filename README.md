# Panda E-Mart — Premium E-commerce Store

A modern, conversion-focused e-commerce website inspired by [http://localhost:3000](https://http://localhost:3000/), built with Next.js App Router, Tailwind CSS, and React.

## Features

- **Premium Design**: Clean, modern UI with Panda E-Mart branding, strong visual hierarchy, and conversion-focused sections
- **Full Commerce**: Product browsing, filtering, search, cart, checkout, and account management
- **Mobile-First**: Responsive design with hamburger menu and app-style bottom navigation
- **Conversion Sections**: Hero, categories, featured products, testimonials, FAQ, trust badges, and newsletter
- **Persistent Cart**: LocalStorage-based cart with backend order integration

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000)

## Project Structure

- `app/` — App Router pages and layouts
- `components/` — UI, layout, home, and commerce components
- `context/` — React context providers (cart, auth, core data)
- `hooks/` — Custom React hooks
- `lib/` — Utility functions
- `public/` — Static assets
